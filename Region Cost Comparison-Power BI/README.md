# Project: Region Cost Comparison

## 📌 Overview
This project focuses on comparing **Azure service costs across multiple regions** using **Azure Cost Management** and visualizing the results in **Excel and Power BI**.  
The objective is to understand how regional pricing differences impact overall cloud spend and support cost‑optimized region selection decisions.

---

## 🏗️ Architecture
Azure cost data is extracted from Cost Management, exported as a CSV file, and analyzed using data visualization tools.

Azure Cost Management  
→ Cost Analysis (Region-based grouping)  
→ CSV Export  
→ Excel / Power BI  
→ Cost Comparison Visualizations

---

## 🔧 Azure Services & Tools Used
- Azure Cost Management
- Azure Cost Analysis
- Microsoft Excel
- Power BI Desktop

---

## ⚙️ Implementation Steps
1. Accessed Azure Cost Management at subscription scope.
2. Analyzed cost data using Cost Analysis.
3. Grouped cost by Azure Region (Location).
4. Exported region-wise cost data as CSV.
5. Imported the data into Excel and Power BI.
6. Created visualizations to compare costs across regions.
7. Identified region-wise cost patterns and cost drivers.

---

## ✅ Outcome
- Successfully compared Azure costs across regions.
- Built Power BI visualizations for region-wise cost analysis.
- Identified cost differences and service-level cost drivers.
- Demonstrated FinOps and cost-optimization analysis skills.
