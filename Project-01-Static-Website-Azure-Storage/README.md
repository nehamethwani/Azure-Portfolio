# Project 1: Static Website Hosting on Azure Storage

## 📌 Overview
This project demonstrates hosting a static website using Azure Blob Storage with static website hosting enabled. It highlights a serverless, cost-effective approach for hosting frontend-only applications.

---

## 🏗️ Architecture
User Browser → Azure Storage Account → Blob Static Website ($web container)

---

## 🔧 Azure Services Used
- Azure Storage Account
- Azure Blob Storage
- Blob Static Website feature

---

## ⚙️ Implementation Steps
1. Created an Azure Storage Account
2. Enabled static website hosting
3. Uploaded HTML and CSS files to the `$web` container
4. Accessed the website via the primary endpoint URL

---

## 🔐 Security & Governance Considerations
- No server or OS exposure
- Read-only public access to static content
- Suitable for non-sensitive frontend content

---

## ✅ Outcome / Learning
- Learned serverless website hosting using Azure Storage
- Understood limitations and use cases of static websites
- Compared static hosting with App Service and VM-based hosting
