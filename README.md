# Azure Portfolio Projects

This repository contains hands-on Azure projects demonstrating practical experience with:
- Infrastructure as a Service (IaaS)
- Platform as a Service (PaaS)
- Azure Storage, App Service, Virtual Machines, and Azure SQL Database
- Enterprise security, networking, and governance constraints

All projects were implemented in a policy-restricted enterprise Azure environment, closely simulating real-world cloud scenarios.

---

## 📂 Projects Included

1. **Static Website Hosting using Azure Storage**
2. **Linux Virtual Machine Web Hosting**
3. **Azure App Service (PaaS) Web Application**
4. **Azure App Service integrated with Azure SQL Database**

Each project folder contains:
- Architecture explanation
- Azure services used
- High-level implementation steps
- Security and governance considerations
- Key learnings

---

## 🧠 Key Takeaways
- Experience with both IaaS and PaaS models
- Understanding of Azure Policies and enterprise governance
- Secure configuration and access management
- Real-world troubleshooting and architectural decision-making
